# music

> myMusic
目前实现的功能有：

    1，热门歌单推荐，最新歌单推荐
    
    2，最新歌曲推荐
    
    3，歌曲搜索，搜索结果：歌单，单曲
    
    4，歌曲播放，播放列表创建
    
    5，歌曲播放模式，目前有：顺序，循环，随机，单曲
    
    6，基于热门歌单和最新歌单获取更多歌单，每次30首
    
    7，可根据搜索结果直接进入对应页面
